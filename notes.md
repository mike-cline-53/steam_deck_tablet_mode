# Steam Deck Project Notes

## Hardware Reference

Official Valve Steam Deck CAD files (CC BY-NC-SA 4.0):
<https://gitlab.steamos.cloud/SteamDeck/hardware>

Includes STEP/CAD models for both LCD and OLED models. Useful for designing
accessories, mounts, cases, and understanding port/button placement.

## Display Specs

| Model | Resolution  | Panel      |
|-------|-------------|------------|
| LCD   | 1280 x 800  | 7" IPS LCD |
| OLED  | 1280 x 800  | 7.4" OLED  |

Portrait mode (rotated): 800 x 1280

## Port / Button Layout (Landscape Orientation)

**Top edge:** USB-C (center), 3.5mm headphone jack, power button, volume +/-
